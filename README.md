# 📚 Student Book Share - Static PDF & PPT Downloader

A modern, fast, and fully responsive single-page static website designed for students to easily search, view, and download academic resources in PDF and PPT formats. The platform also allows students to contribute by uploading their own study materials.

## ✨ Features
* **Instant Search & Filter:** A dynamic search bar that filters books instantly by title, subject, or author without reloading the page.
* **Modern UI Grid Layout:** Clean, beautifully animated card layouts for every book with visual format tags (PDF/PPT).
* **Instant Downloads:** Direct links to access or save study materials.
* **Contribution Form:** An interactive upload section where students can contribute book details and files to expand the library.
* **Fully Responsive:** Optimized for a seamless experience on mobile phones, tablets, and desktop computers.

## 📁 Project Structure
The project is built entirely using clean, semantic, and modern standard web technologies across three separate files:
```text
├── index.html   # Semantic markup and structure of the application
├── styles.css   # Modern CSS Grid/Flexbox styling, custom properties, and UI transitions
└── script.js     # Client-side JavaScript handling instant search filtering and upload simulations
```

## 🛠️ Tech Stack
* **HTML5:** Semantic architecture for accessibility.
* **CSS3:** Modern grid layout, flexbox, variable themes, and smooth animations.
* **JavaScript (ES6):** Client-side DOM manipulation, string matching algorithms for search, and event handling.

## 📝 Future Scope
* Integration with a backend database (like Firebase or Supabase) to permanently store and approve user-uploaded files.
* Dark mode toggle for nighttime studying.
